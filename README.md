# StudioFlow

StudioFlow is a premium service studio management platform for tattoo studios, salons, barber shops, piercing studios, wellness clinics, and solo appointment-based businesses.

The product is designed around a modern booking-first workflow with a polished SaaS dashboard experience.

## Features

- Role-based dashboards for admin, staff, receptionist, and customer views
- Calendar-driven appointment management
- Client records and visit history
- Staff and service management
- Payments and deposit tracking
- Consent forms and document workflows
- Business analytics surfaces

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- React Router
- Vite

## Scripts

- `npm run dev` starts the local development server
- `npm run build` creates the production build
- `npm run lint` runs ESLint
- `npm run deploy` publishes the app to GitHub Pages

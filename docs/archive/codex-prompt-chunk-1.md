# Chunk 1 - Project Overview Prompt

Build a modern full-stack service studio management platform called StudioFlow for businesses like tattoo studios, salons, barber shops, piercing studios, and wellness providers.

The product should feel like a premium SaaS dashboard, with the calendar and appointment management system as the main core feature.

## Design direction

- Inspired by modern booking platforms like Fresha
- Do not clone exactly
- Create an original UI with cleaner spacing, more premium typography, and better SaaS polish
- Main layout should use:
  - dark left sidebar
  - light main content area
  - top control bar
  - rounded cards
  - soft shadows
  - muted appointment colors
- UI should feel:
  - professional
  - fast
  - trustworthy
  - modern
  - responsive

## Tech stack

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- React Router

## Design rules

- Use reusable components
- Mobile responsive
- Clean spacing
- Premium dashboard feel
- No loud colors
- Use muted teal, blue, peach, lavender, and rose for booking cards
- Use dark charcoal or navy for the sidebar
- Use off-white or light gray for the background

# StudioFlow Application Snippets

## Recruiter-Friendly One-Sentence Description

StudioFlow is a full-stack SaaS platform for service businesses that combines internal operations workflows with customer-facing booking and self-service management.

## Two-Sentence Description

StudioFlow is a multi-location SaaS platform built for appointment-based service businesses such as salons, tattoo studios, barbershops, and wellness practices. It includes an internal workspace for scheduling, clients, staff, payments, consent forms, notifications, analytics, and audit history, plus a separate public booking and customer self-service flow.
